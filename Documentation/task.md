Проект: Система мониторинга цен конкурентов с AI-аналитикой
🧠 Описание проекта
Автоматизированная система мониторинга цен конкурентов. Парсит сайты, отслеживает изменения цен, анализирует тренды с помощью AI и отправляет уведомления о важных изменениях.
🎯 Цель
Научиться создавать парсеры, работать с прокси, анализировать временные ряды и строить рекомендательные системы на основе AI.
📌 Этапы выполнения
Этап 1: Анализ и проектирование
	• Выбрать сайты для парсинга
	• Изучить структуру страниц
	• Спроектировать парсеры и базу данных
Этап 2: Разработка парсеров
	• Создать пауки для каждого сайта
	• Реализовать обход защиты
	• Настроить сохранение в базу данных
Этап 3: Планировщик задач
	• Настроить автоматический запуск парсинга
	• Реализовать очередь задач
	• Добавить логирование и обработку ошибок
Этап 4: AI-анализ
	• Анализ ценовых трендов
	• Создание промптов для рекомендаций
	• Генерация советов по ценообразованию
Этап 5: Frontend-дашборд
	• Создать интерфейс с графиками
	• Реализовать таблицу товаров с фильтрами
	• Добавить настройки парсинга
Этап 6: Система уведомлений
	• Telegram-бот для алертов
	• Настройка правил уведомлений
	• История изменений цен
Этап 7: Деплой
	• Развернуть систему на сервере
	• Настроить мониторинг
Оптимизировать производительность
